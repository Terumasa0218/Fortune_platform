# 占術計算方式の仕様

本文書は Fortune Platform で採用する各占術の計算方式を定義する。
実装時はこの仕様に厳密に従うこと。

---

## 1. 西洋占星術（Western Astrology）

### 1.1 黄道帯方式
- **トロピカル（Tropical）** を採用
- 春分点を牡羊座0度に固定
- 歳差運動は考慮しない

### 1.2 ハウスシステム
- **Placidus** を採用
- 最も主流な方式であり、多くの占星術ソフトでデフォルト

### 1.3 使用天体
以下の10天体 + 計算点を使用:
- **古典7天体**: 太陽、月、水星、金星、火星、木星、土星
- **現代3天体**: 天王星、海王星、冥王星
- **計算点**: ASC（Ascendant）、MC（Medium Coeli）

### 1.4 アスペクト
#### メジャーアスペクト（±8°）
- 0° Conjunction（合）
- 60° Sextile（六分）
- 90° Square（矩）
- 120° Trine（三分）
- 180° Opposition（衝）

#### マイナーアスペクト（±2°）
- 30° Semi-sextile
- 45° Semi-square
- 135° Sesquiquadrate
- 150° Quincunx

## 2. インド占星術（Vedic Astrology）

### 2.1 アヤナムシャ（歳差補正）
- **Lahiri** を採用
- インド政府公式採用の方式
- 2000年時点で約24度の補正値

### 2.2 ハウスシステム
- **Whole Sign** を採用
- ラグナ（ASC）のある星座全体を第1ハウスとする伝統的方式

### 2.3 相性判定方式（Phase 2E実装時に詳細化）
- Ashtakoota（8要素）またはGun Milanを採用予定
- 月の位置（Nakshatra）を重視

---

## 3. 四柱推命（Four Pillars of Destiny）

### 3.1 節入り基準
- **立春基準** を採用
- 毎年2月4日前後の立春を年の起点とする
- 他の節気（啓蟄、清明等）も正確な時刻で判定

### 3.2 真太陽時補正
- **補正する**
- 各都市の経度差を反映し、地方真太陽時を計算
- 時柱の正確な干支判定に必要

### 3.3 構成要素
- 年柱、月柱、日柱、時柱の干支
- 通変星、十二運、五行バランス

---

## 4. 紫微斗数（Zi Wei Dou Shu）

### 4.1 流派
- **中州派** を採用
- 最も主流で体系化されている流派

### 4.2 命盤構成
- 12宮（命宮、兄弟宮、夫妻宮、子女宮、財帛宮、疾厄宮、遷移宮、奴僕宮、官禄宮、田宅宮、福徳宮、父母宮）
- 主星14個 + 副星・雑曜

---

## 5. 性格診断系（MBTI / エニアグラム / ラブタイプ）

### 5.1 実装方式
**外部診断結果の手動入力のみを採用**
- プラットフォーム内での質問フローは実装しない
- ユーザーが外部サイト（16Personalities等）で診断した結果を入力する形式

### 5.2 MBTI（Myers-Briggs Type Indicator）
- **入力方式**: 16タイプ（ENFP, ISTJ, ... 等）をドロップダウンで選択
- **推奨診断サイト**: 16Personalities (https://www.16personalities.com)
- **相性判定**: 16タイプ × 16タイプ の相性マトリクスを用意
- **評価軸**: コミュニケーション、衝突、安定性、成長の4軸

### 5.3 エニアグラム（Enneagram）
- **入力方式**: タイプ1〜9をドロップダウンで選択
- **ウィング**: タイプのみ（ウィングは考慮しない、Phase後期で追加可）
- **相性判定**: 9タイプ × 9タイプ の相性マトリクス

### 5.4 ラブタイプ（Love Type）
- **入力方式**: 独自定義の5〜8タイプをドロップダウンで選択
- **タイプ定義**: Phase 2B実装時に決定
- **相性判定**: タイプ間の相性マトリクス

## 実装優先順位

Phase 2 の実装順序:
1. **Phase 2A**: タロット（実装済み）
2. **Phase 2B**: MBTI + エニアグラム + ラブタイプ（手動入力、計算不要）
3. **Phase 2C**: 西洋占星術（メイン機能、天体計算必要）
4. **Phase 2D**: 四柱推命
5. **Phase 2E**: インド占星術
6. **Phase 2F**: 紫微斗数
